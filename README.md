# Learnings-Geron-course-on-ML
This is exclusively for learning purpose using the Hands-On ML by Geron. The objective is to have a hands on the main machine learning algorithm as well as analytical thinking leading to good model for decision-making.

Therefore, the subjects to cover here are:
  1. Data manipulation and visualization
  2. Data Engineering: pipeline approach
  3. Solving a regression problem with simple and more advanced models
  4. Solving classification problems
